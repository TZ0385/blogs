# 司法审查

司法审查涉及法律领域的网络安全相关问题，包括电子证据的法律效力、网络安全事件的报告义务、以及执法部门的配合要求。本文档旨在帮助安全从业者了解相关法律要求。

## 1 法律框架

### 1.1 关键法律

| 法律 | 适用范围 | 关键要求 |
|------|----------|----------|
| 《网络安全法》 | 全行业 | 等级保护、安全评估、数据保护 |
| 《数据安全法》 | 数据处理者 | 数据分类分级、应急响应 |
| 《个人信息保护法》 | 个人信息处理 | 隐私保护、用户权利 |
| 《刑法》第285/286条 | 入侵计算机系统 | 非法侵入、破坏计算机系统罪 |
| 《电子签名法》 | 电子证据 | 电子签名法律效力 |

### 1.2 网络安全等级保护 (等保)

```text
等保2.0分级:
├── 第一级 (自主保护)
│   └── 适用于小型私营企业
│
├── 第二级 (指导保护)
│   ├── 50人以下运营者
│   ├── 非关键系统
│   └── 需要备案
│
├── 第三级 (监督保护)
│   ├── 关键基础设施
│   ├── 100万用户数据
│   └── 每年安全评估
│
├── 第四级 (强制保护)
│   └── 政府、军事、重要行业
│
└── 第五级 (专控保护)
    └── 国防、极端关键系统
```

## 2 电子证据

### 2.1 电子证据的法律地位

```text
电子证据要求:
├── 真实性
│   ├── 证据来源可靠
│   ├── 内容未被篡改
│   └── 存储介质可靠
│
├── 完整性
│   ├── 数据完整未丢失
│   ├── 哈希校验一致
│   └── 形成时间可追溯
│
├── 合法性
│   ├── 收集程序合法
│   ├── 未侵犯隐私
│   └── 符合证据规则
│
└── 关联性
    ├── 与案件相关
    └── 能够证明事实
```

### 2.2 电子证据收集

```cmd
# 1. 硬盘镜像
# 使用FTK Imager或dd
ftkimager --verify D:\: evidence.E01 --compress 9

# 2. 内存获取
# 使用WinPcap或DumpIt
# 必须在关机前获取内存

# 3. 日志获取
# 导出Windows事件日志
wevtutil epl Security C:\temp\security.evtx
wevtutil epl System C:\temp\system.evtx

# 4. 计算哈希
certutil -hashfile "C:\evidence\image.bin" SHA256
certutil -hashfile "C:\evidence\image.bin" MD5
```

### 2.3 证据保全

```powershell
# 1. 哈希计算
Get-FileHash -Algorithm SHA256 -Path C:\temp\evidence.bin

# 2. 写保护
# 使用硬件写保护接口
# 启用只读模式

# 3. 证据链记录
# 记录: 谁、什么时间、什么操作、什么工具
# 记录格式:
# -------------------------------------------------
# 证据编号: EV-2024-001
# 获取时间: 2024-01-15 14:30:00 CST

# 工具: FTK Imager v4.2.0

# 4. 存储
# 原件: 密封存储于保险柜
```

## 3 网络安全事件分类

### 3.1 事件分级

```text
网络安全事件分级:
├── 特别重大事件 (一级)
│   ├── 全国性骨干网络中断
│   ├── 多个省份重要系统瘫痪
│   └── 影响国家安全事件
│
├── 重大事件 (二级)
│   ├── 单个省区网络中断
│   ├── 重要信息系统严重受损
│   └── 大量用户数据泄露
│
├── 较大事件 (三级)
│   ├── 单个地市网络中断
│   ├── 系统业务中断超24小时
│   └── 部分用户数据泄露
│
└── 一般事件 (四级)
    ├── 单个系统受影响
    ├── 短暂服务中断
    └── 个别用户受影响
```

```powershell
Get-ADGroupMember "Domain Admins" -Recursive

# 2. 日志审计
Get-WinEvent -FilterHashtable @{LogName='Security';ID=4624} -MaxEvents 100 | Select-Object TimeCreated, Message

# 4. 补丁审计
```
