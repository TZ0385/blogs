# 内网扫描探测

内网扫描探测是在获得内网入口后，对目标网络进行全面探测，发现存活主机、开放端口和服务信息，为后续横向移动做准备。

## 1 内网探测基本流程

```
内网探测流程:
├── 存活主机发现
│   ├── ARP扫描
│   ├── ICMP扫描
│   ├── TCP/UDP探测
│   └── NetBIOS探测
│
├── 端口扫描

    ├── 资产清单
    ├── 拓扑绘制
    └── 攻击路径规划
```

## 2 存活主机发现

### 2.1 ARP扫描

```bash
# arp-scan (推荐)
arp-scan -l
arp-scan --localnet
arp-scan 192.168.1.0/24

# arp -a (Windows)
arp -a
arp -a 192.168.1.0

# 主动解析
for /L %i in (1,1,254) do @ping -n 1 -w 100 192.168.1.%i | find "TTL"

# PowerShell
1..254 | ForEach-Object { $ip="192.168.1.$_"; if(Test-Connection -ComputerName $ip -Count 1 -Quiet) { $ip } }

# netdiscover (被动)
netdiscover -p
netdiscover -i eth0 -r 192.168.1.0/24
```

### 2.2 ICMP扫描

```bash
# ping 批量扫描
for i in {1..254}; do ping -c 1 -W 1 192.168.1.$i & done | grep "64 bytes"

# fping
fping -g 192.168.1.0/24
fping -a -g 192.168.1.0/24 2>/dev/null

# nmap ICMP扫描
nmap -sn -PE -T4 192.168.1.0/24
nmap -sn -PP -PM 192.168.1.0/24  # PE=echo, PM=mask

# hping3
hping3 -1 -c 3 192.168.1.1
hping3 -1 -dead 192.168.1.0/24
```

### 2.3 NetBIOS扫描

```bash
# nbtscan (Kali自带)
nbtscan -r 192.168.1.0/24
nbtscan -v -s : 192.168.1.0/24

# Windows自带
nbtstat -A 192.168.1.1

# enum4linux
enum4linux -a 192.168.1.1
enum4linux -S 192.168.1.0/24
```

## 3 端口扫描

### 3.1 Nmap扫描

```bash
# 快速扫描常用端口
nmap -sT --top-ports 100 -T4 192.168.1.0/24

# SYN扫描 (需要root权限)
nmap -sS -T4 192.168.1.0/24

# UDP扫描
nmap -sU -T4 192.168.1.0/24

# 全端口扫描
nmap -sT -p- -T4 192.168.1.1
nmap -sS -p 1-65535 -T4 192.168.1.1

# 绕过防火墙
nmap -sS -T4 -f 192.168.1.1  # 碎片化
nmap -sS --data-length 50 -T4 192.168.1.1  # 数据填充
nmap -sS --badsum -T4 192.168.1.1  # 错误校验

# 版本检测
nmap -sV -sS -T4 192.168.1.0/24

# 操作系统检测
nmap -sS -O -T4 192.168.1.1

# 脚本扫描
nmap -sS --script=vuln 192.168.1.0/24  # 漏洞扫描
nmap -sS --script=smb-vuln* 192.168.1.0/24  # SMB漏洞
nmap -sS --script=ssh-brute 192.168.1.1  # SSH暴力破解

# 常用Nmap参数
# -Pn: 不ping，直接扫描
# -n: 不做DNS解析
# -R: 总是做DNS解析
# -oA: 输出所有格式
```

### 3.2 Masscan (高速扫描)

```bash
# 安装 (Kali自带)
# 使用示例
masscan -p1-65535 192.168.1.0/24 --rate=10000 -oJ result.json

# 常用参数
masscan -p80,443,22,3389 192.168.1.0/24 --rate=1000
masscan -p1-1000 10.0.0.0/8 --rate=10000
```

### 3.3 内网扫描脚本

```bash
# PowerShell 批量端口扫描
$ports = @(21,22,23,25,80,110,135,139,443,445,1433,1521,3306,3389,5432,6379,8080,27017)
$ips = 1..254 | ForEach-Object { "192.168.1.$_" }
foreach ($ip in $ips) {
    foreach ($port in $ports) {
        $tcp = New-Object System.Net.Sockets.TcpClient
        try {
            $tcp.Connect($ip, $port)
            Write-Host "$ip`:$port is OPEN"
            $tcp.Close()
        } catch {}
    }
}
```

## 5 漏洞扫描


## 6 域环境探测

### 6.1 域信息收集

```powershell
# 查看当前域
Get-NetDomain
(Get-ADDomain).DNSRoot

# 查看域控制器
Get-NetDomainController
Get-ADDomainController -Discover

# 查看域用户
Get-NetUser
Get-ADUser -Filter * | Select-Object Name,SamAccountName

# 查看域计算机
Get-NetComputer
Get-ADComputer -Filter * | Select-Object Name,OperatingSystem

# 查看域管用户
Get-NetGroup "Domain Admins"
Get-ADGroupMember "Domain Admins" -Recursive

# 查看域策略
Get-DomainPolicy
(Get-DomainPolicy)."SystemAccess"
```

### 6.2 域信任关系

```powershell
# 查看域信任
Get-NetDomainTrust
Get-ADTrust -Filter *

# 查看林信任
Get-NetForest
Get-NetForestTrust
```

## 10 防御措施

```bash
# 1. 网络分段
#    - VLAN划分
#    - 子网隔离

# 2. 防火墙规则
#    - 禁用不必要的端口
#    - 限制ICMP流量
#    - 启用IDS/IPS

# 3. 监控
#    - 部署网络监控
#    - 日志审计
#    - 流量异常检测

# 4. 主机安全
#    - 关闭不必要的服务
#    - 定期打补丁
#    - 启用防火墙

# 5. 域安全
#    - 限制域管登录范围
#    - 启用 SMB 签名
#    - 监控异常登录
```
